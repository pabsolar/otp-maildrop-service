# OTP Maildrop Service

## Project Description
This project is a Spring Boot 3.5 application that provides an OTP (One-Time Password) mail drop service. The service is designed to securely facilitate the generation, storage, and email delivery of OTPs for authentication and verification purposes. 

### Features
- Generate OTPs for user verification.
- Store OTPs securely.
- Send OTPs to users via email.
- RESTful API for integration with other services.

### Technologies Used
- Spring Boot 3.5
- Java
- Maven
- JPA
- Hibernate
- SMTP for email

### Getting Started
1. Clone the repository.
2. Configure the application settings in `application.properties`.
3. Run the application using your preferred method (e.g., IDE, Maven).
